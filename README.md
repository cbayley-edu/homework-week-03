# 03 JavaScript: Password Generator

## User Story

```
AS AN employee with access to sensitive data
I WANT to randomly generate a password that meets certain criteria
SO THAT I can create a strong password that provides greater security
```

## Instructions

```
When launching the website, click the Generate Password button.
Follow the prompts to:
  * enter the desired password length (8 - 128 characters)
  * select password components - lowercase, uppercase, numbers, special characters
Exceptions - you will need to restart:
  * if you enter an invalid number or no number
  * if you do not select OK for at least one of the four prompts
Upon completing the prompts, your new password will appear in the text box in the center of the screen.
Select the text and use ctrl+C to copy the generated password
```
